Visual Studio Code is an IDE, similar to Brackets, but the layout is a little different. It is made by Microsoft, and can be used on any device. You can download and install it at this link



The Console can be used to log information, as well as allow you to interact with a web page by carrying out JavaScript expressions within the page’s context. The console allows you to keep track of what is going on in your code. This makes it easy to figure out what’s wrong in your code.
